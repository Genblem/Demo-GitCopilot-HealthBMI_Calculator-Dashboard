BMI Calculator Demo
===================

This is a simple BMI Calculator web application built with HTML, Tailwind CSS, JavaScript, and SweetAlert2.

Features:
- Login system (username: demo, password: dome)
- BMI calculation with result status and health advice
- Shows daily calorie, water, BMR, body fat %, and protein recommendations
- Responsive, modern UI with Tailwind CSS
- Logout and fade-in animation

How to use:
1. Clone or download this repository.
2. Open `src/index.html` in your browser.
3. Login with username: demo and password: dome
4. Enter your weight, height, age, and gender, then click Calculate BMI.

For development/testing:
- All calculation logic is in `src/app.js` and can be tested with Jest (`npm test`).
